Armed with nothing but qemu, I set about rediscovering the joy of mindless tinkering by installing 10 distros in 10 days. This is what I found.

the host

the most boring part is upfront, preparing the host machine. luckily, there is only one thing to install — qemu 

brew install qemu

well, that was easy.

the machine

if we install an OS in a qemu machine (“VM”) that has the same CPU architecture as our host machine, it’ll run at ~native speed (that how clouds work, theyqemutoo). I also want nice graphics, and sound. After some experimentation, I arrive at a qemu invocation that I then put in a simple shell script.
